How to Compile and Run:
* The linux utility make should be installed, otherwise should be compiled manually.
*Just type 'make' from the this directory.
*The 'make' command will generate the program with name 'program' in this directory.
*programs takes three command line arguments, start, end and k. start and end are for range of n, and are incremented by 5. The value of k is not incremented.
*The outputs after running './program' from this directory will be in csv format,one is 'running-time.csv' and the other is 'n-subproblems.csv' for running time and number of subproblems solved for three algorithms with varying N.
*Those csv files can be used for analysis and plotting the running time of the program.
